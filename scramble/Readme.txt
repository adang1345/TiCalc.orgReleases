Scramble 4.0 by Aohan Dang

Description
Generate a random sequence of up to 54 moves to scramble a Rubik's Cube.

How to install and run
Send scram.89p and exprcomb.89f to your TI-89 or TI-89 Titanium. To run, type scram() on the
home screen. To quit, enter the number of moves as 0. A side effect of the program is it sets
the "Exact/Approx" mode to "EXACT."
