Find Surface Area by Aohan Dang

Description
Find the surface area of spheres, cylinders, and cones.

How to install and run
Send surfarea.89p to your TI-89 or TI-89 Titanium. Type surfarea() on the home screen and press
ENTER. After selecting which shape with the F2 through F4 keys, input the length of the dimension
for which the calculator asks.
