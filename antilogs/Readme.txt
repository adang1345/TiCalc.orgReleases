Antilogarithmic Functions by Aohan Dang

Description
Find antilogarithms with base 10, base e, or a specified base.

How to install and use
Send antiln.89f, antilogb.89f, and antilog.89f to your TI-89 or TI-89 Titanium.

To find the anti-natural logarithm of a number:
antiln(number)

To find the antilogarithm of a number with base 10:
antilog(number)

To find the antilogarithm of a number to any base:
antilogb(number,base)
