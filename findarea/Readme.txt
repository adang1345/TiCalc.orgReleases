Find Area by Aohan Dang

Description
Find the area of triangles, trapezoids, circles, and ellipses.

How to install and run
Send area.89p to your TI-89 or TI-89 Titanium. Type area() on the home screen and press	ENTER.
After selecting which shape with the F2 through F5 keys, input the length of the dimension for
which the calculator asks.
