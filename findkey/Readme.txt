Findkey by Aohan Dang

Description
The calculator thinks of a key (without use of modifier keys) for you to guess.

How to install and play
Send findkey.89p to your TI-89 or TI-89 Titanium. Type findkey() on the home screen. Without
using modifier keys, try to guess the key of which the calculator is thinking. Press the key to
see if you are correct.
