Do Not Press Enter by Aohan Dang

Description
A model of Do Not Push the Red Button. The calculator says DO NOT PRESS ENTER, and of course
you will.

How to install and run
Send dontpres.9xz to your TI-92 Plus. Type dontpres() on the home screen. Warning-
this program is long and you can't end the program in the middle except by resetting your
calculator. If you don't have 5 minutes to spare, don't run this program.
