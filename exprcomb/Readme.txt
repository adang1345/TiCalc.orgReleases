Expression Combine by Aohan Dang (adang1345@gmail.com)

Description
Combine two expressions into one expression. For example, a and 1 becomes a1.

How to install and run
Send exprcomb.89f to your TI-89 or TI-89 Titanium. The correct syntax for this function is
exprcomb(expression,expression). For this function to work, the "Exact/Approx" mode must be
either set as "EXACT" or "AUTO."
