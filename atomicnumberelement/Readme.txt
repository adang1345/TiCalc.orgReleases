Atomic Number <--> Element Name by Aohan Dang

Description
Enter the atomic number and find the name of the element or enter the element and find its
atomic number.

How to install and run
Send element.89f to your TI-89 or TI-89 Titanium. The correct syntax for converting atomic
number to element is element(number). The number can be an integer from 1 to 118. The syntax
for converting element name to atomic number is element("name of element"). Make sure you use
lowercase letters and spell the element name correctly or the function does not work properly.
