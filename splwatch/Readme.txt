Split Time Stopwatch 1.0 by Aohan Dang

Description
Start a stopwatch and save split times.

How to install and run
Send splwatch.89p to your TI-89 or TI-89 Titanium. Type splwatch() on the home screen and press
ENTER. Use the toolbar to start the timer. After you are done recording all your split times,
the calculator displays them. Press ENTER to view the next split time.
