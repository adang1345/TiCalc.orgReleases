Unweighted GPA Calculator 1.0 by Aohan Dang

Description
Find your unweighted GPA by inputting the number of courses you take and the grades you received
in each.

How to install and run
Send ugpa.89p to your TI-89 or TI-89 Titanium. Type upga() on the home screen and press ENTER.
When the calculator asks for you to input grades, valid inputs are A+, A, A-, B+, B, B-, C+, C,
C-, D+, D, D-, and F.
