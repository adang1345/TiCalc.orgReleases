Teach Chat 1.0 by Aohan Dang

Description
Chat with the calculator by teaching it possible statements. The program saves what you input
and displays a random response chosen from what you inputted. Entertain yourself with responses
that make sense and responses that are stupid. You can save and load the database of possible
responses.

How to install and run
Send teachat.89p to your TI-89 or TI-89 Titanium. Type teachat() on the home screen and press
ENTER. To access the menu, press ON. For best experience with this program, you should be
experienced in typing on the TI-89 Titanium keypad. A good program to help you get better at
typing is Type Train. Download it if you wish.
