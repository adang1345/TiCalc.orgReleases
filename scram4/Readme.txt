Rubik's 4x4 Scrambler 1.0 by Aohan Dang

Description
Generate a random sequence of up to 24 moves to scramble a Rubik's 4x4 cube.

How to install and run
Send scram4.89p to your TI-89 or TI-89 Titanium. Type scram4() on the home screen.
Be patient when running this program for the first time. It takes a while to load,
but the next time you run it, loading time is fast.
