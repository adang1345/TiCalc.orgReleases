Scram67 3.0 by Aohan Dang

Description
Generate a random sequence of up to 48 moves to scramble a V-Cube 6 or V-Cube 7.

How to install and run
Send scram67.89p and exprcomb.89f to your TI-89 or TI-89 Titanium. Type scram67() on the
home screen and press ENTER. To quit, enter the number of moves as 0.  A side effect of the program is it sets
the "Exact/Approx" mode to "EXACT."
