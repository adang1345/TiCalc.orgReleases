Cmd 1.0 by Aohan Dang

Description
A version of command prompt for the TI-89.

How to install and run
Send cmd.89p to your TI-89 or TI-89 Titanium. Type cmd() on the home screen and press ENTER.
Type in a command for the program to execute. When execution of the command finishes, the program
returns to its original state. To close, enter the command close. If an error occurs while
executing the command, an error message pops up, and the program returns to its original state.
