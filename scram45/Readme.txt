Rubik's 4x4 and 5x5 Scrambler 4.0 by Aohan Dang

Description
Generate a random sequence of up to 54 moves to scramble a Rubik's 4x4 or 5x5.

How to install and run
Send scram45.89p and exprcomb.89f to your TI-89 or TI-89 Titanium. Type scram45() on the
home screen and press ENTER. To quit, enter the number of moves as 0.  A side effect of the program is it sets
the "Exact/Approx" mode to "EXACT."
