Find Volume by Aohan Dang

Description
Find the volume of prisms, pyramids, cones, and spheres.

How to install and run
Send volume.89p to your TI-89 or TI-89 Titanium. Type volume() on the home screen and press ENTER.
After selecting which shape with the F2 through F5 keys, input the length of the dimension for
which the calculator asks.
